Se o CUPS “perder” a impressora (raro, mas acontece)
Remova e adicione de novo:

sudo lpadmin -x MG2410

Depois reconfigure (USB):
sudo lpadmin \
  -p MG2410 \
  -E \
  -v usb://Canon/MG2410 \
  -m everywhere


Se quiser confirmar o URI correto:
lpinfo -v

Dica importante (evita problemas no futuro)
Configure o CUPS para não desabilitar a impressora automaticamente quando ela desligar:

sudo lpadmin -p MG2410 -o printer-error-policy=retry-job


Reativar a impressora mais comum
sudo cupsenable MG2410
sudo cupsaccept MG2410


Antes de imprimir agora defina a impressora como padrão:
sudo lpadmin -d MG2410


Utilizando a impressora para fazer uma copia de teste: verificar se está funcionando
echo "Teste" | lp -d MG2410
